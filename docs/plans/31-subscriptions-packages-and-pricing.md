# 31. Подписки и цены (этап 1)

## Общие условия

- Бот включен во все подписки.
- На этапе 1 только WhatsApp (без Telegram).
- Во всех подписках доступны:
  - редирект/эскалация на WhatsApp администратора;
  - подтверждение записи с WhatsApp администратора.

## Starter

- Цена: `€59 / месяц`
- Салоны: до 1
- Мастера: до 3
- Лимит записей: до ~300 / месяц

## Growth

- Цена: `€109 / месяц`
- Салоны: до 1
- Мастера: до 7
- Лимит записей: до ~1000 / месяц
- Дополнительно: напоминания `24h/2h`, расширенная аналитика, приоритетная поддержка

## Pro

- Цена: `€209 / месяц`
- Салоны: до 3
- Мастера: по согласованию
- Лимит записей: до ~3000 / месяц (суммарно)
- Дополнительно: роли команды, продвинутые FAQ/AI настройки, SLA

## Enterprise

- Цена: индивидуально
- Салоны: 4+
- Лимиты: индивидуально
- Возможности: кастомные интеграции, расширенный SLA

## Управление из супер-админки

- Цена тарифа должна редактироваться из админки без деплоя.
- Состав услуг и лимитов пакета должен редактироваться из админки.
- Источник истины: БД (версионные опубликованные тарифы), markdown-файл — продуктовая фиксация.

## Примечание по WhatsApp стоимости

- Стоимость template-сообщений Meta считается отдельно по действующим ставкам страны.
- Для биллинга можно выбрать:
  - пакет сообщений в подписке + overage;
  - pass-through отдельной строкой.
