# Genius Clients Logo Pack

Files:
- `logo-horizontal.svg` - primary logo for website header, презентации, and business cards.
- `logo-mark.svg` - standalone icon mark for app icon, social avatar, or compact placements.
- `favicon.svg` - browser favicon source.

Recommended usage:
- Website header: `logo-horizontal.svg` (height 32-48px).
- Favicon: `favicon.svg` (and optionally export 32x32 PNG/ICO for legacy browsers).
- Business cards: `logo-horizontal.svg` for front side, `logo-mark.svg` for back side.

Color direction:
- Teal primary + soft lime accent, matching the current product palette.
